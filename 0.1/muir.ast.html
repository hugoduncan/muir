muir.ast documentation

AST Transformation

This is a wrapper for the analyze lib, which adds helpers, an open coded AST traversal, and an open coded AST to form function.

deftraversal

macro

[name key-sequence-map pd-keys]

Defines a traversal multi-method. The key-sequence-map is a map from AST node :op to a sequence of keywords, specifying a traversal order for child or child-seq nodes. pd-keys specifies a sequence of parent AST node keys that should be passed down to each child node during the traversal. The generated traversal passes no map to the parent nodes.

update-node

fn

[ast-node kw f pd-keys]

Transforms a child in an AST node. Takes an AST node ast-node, a keyword kw identifying a child in that node, and a transformation function f and a sequence of node member keywords pd-keys. The transformation function is applied to the child node merged with the ast-node keys specified by pd-keys. It is expected to return a vector containing the modified node, and a map which is merged into the parent node.

update-node-seq

fn

[ast-node kw f pd-keys]

Transforms a child sequence in an AST node. Takes an AST node ast-node, a keyword kw identifying a child sequence in that node, and a transformation function f and a sequence of node member keywords pd-keys. The transformation function is applied to the child sequence's nodes merged with the ast-node keys specified by pd-keys. It is expected to return a vector containing the modified node, and a map which is merged into the parent node.
